# Local Skills

このディレクトリは、本プロジェクト専用の再利用可能な作業手順（skill）をまとめる場所です。

## 目的

- エージェント/開発者が同じ流れで作業できるようにする
- IFC 学習用可視化プロジェクトの設計思想を維持する
- 新規参加者の立ち上がりコストを下げる

## 収録ルール

- 1 スキル = 1 フォルダ
- 各フォルダに `SKILL.md` を必須で置く
- 必要に応じて `references/` や `templates/` を追加してよい

## 現在のスキル

- `sample-authoring`: 新しい例題（samples）を追加する手順
- `operation-implementation`: 新しい IFC 形状オペレーションを追加する手順
